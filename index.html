CS Fundamentals Learning Path

1. Programming Fundamentals
▼

What you'll learn:

Master the fundamentals of programming including variables, data types, control structures, functions, and basic problem-solving techniques. Learn multiple programming languages and understand programming paradigms.

PDF

Introduction to Programming

Comprehensive guide covering basic programming concepts with examples in Python, Java, and C++.

VIDEO

Harvard's renowned course covering programming fundamentals and computational thinking.

Quick reference guide with syntax and concepts for popular programming languages.

100+ coding problems to practice basic programming concepts and logic building.

2. Object-Oriented Programming
▼

What you'll learn:

Understand classes, objects, inheritance, polymorphism, encapsulation, and abstraction. Learn to design and implement object-oriented solutions to complex problems using design patterns.

PDF

OOP Principles and Design Patterns

Deep dive into object-oriented design patterns, SOLID principles, and best practices.

VIDEO

Object-Oriented Programming in Java

Complete video series covering OOP concepts with practical examples and projects.

Interactive course covering 23 classic design patterns with real-world implementations.

3. Computer Architecture & Systems
▼

What you'll learn:

Understand how computers work at the hardware level including CPU, memory, storage, and I/O systems. Learn assembly language, computer organization, and system performance optimization.

PDF

Computer Systems: A Programmer's Perspective

Comprehensive textbook covering computer architecture from a programmer's viewpoint.

VIDEO

MIT 6.004 - Computation Structures

MIT's computer architecture course covering digital logic and computer organization.

PRACTICE

Hands-on exercises in x86 assembly language and low-level programming.

4. Operating Systems
▼

What you'll learn:

Master OS concepts including processes, threads, memory management, file systems, and synchronization. Learn Linux/Unix commands and system programming.

PDF

Operating System Concepts

Classic textbook covering all major OS concepts with practical examples.

VIDEO

UC Berkeley CS162 - Operating Systems

Complete operating systems course with lectures and programming assignments.

PRACTICE

Linux System Administration

Practical exercises in Linux commands, shell scripting, and system management.

▼

Master semantic HTML5, modern CSS3 including Flexbox and Grid, responsive design, accessibility principles, and CSS preprocessing with Sass/SCSS.

PDF

▼

What you'll learn:

Master JavaScript fundamentals, ES6+ features, DOM manipulation, event handling, asynchronous programming with promises and async/await, and modern JavaScript tools.

PDF

Master modern frontend frameworks including React with hooks, Vue.js, or Angular. Learn component-based architecture, state management, routing, and testing.

Master server-side programming with Node.js, Python (Django/Flask), or PHP. Learn API development, authentication, middleware, and server deployment.
